対応ボード:
- [Pro Micro](#ボード-sparkfun-pro-micro-atmega32u4-16mhz-5v)
- [UIAPduino](#ボード-uiapduino-atmega32u4-系)

ボード固有の設定は `board/mydeck/board_config.h` に定義されています。
`platformio.ini` の `build_flags` で `-DBOARD_PROMICRO` または `-DBOARD_UIAPDUINO` を選択します。

```

┌─────────────────┐
│ GND  [●]────────┼── ボタン共通 GND
│  2   [●]────────┼── ボタン1
│  3   [●]────────┼── ボタン2
│  4   [●]────────┼── ボタン3
│  5   [●]────────┼── ボタン4
│  6   [●]────────┼── ボタン5
│  7   [●]────────┼── ボタン6
│  8   [●]────────┼── ボタン7
│  9   [●]────────┼── ボタン8
│ 10   [●]────────┼── ボタン9
│ 14   [●]────────┼── ボタン10 (MISO)
│ 15   [●]────────┼── ボタン11 (SCK)
│ 16   [●]────────┼── ボタン12 (MOSI)
│ 18   [●]────────┼── ボタン13 (A0)
│ 19   [●]────────┼── ボタン14 (A1)
│ 20   [●]────────┼── ボタン15 (A2)
│ 21   [●]────────┼── ボタン16 (A3)
│ USB  [■]────────┼── PC USB ポート
└─────────────────┘
```

## 注意事項

- SPI 兼用ピン（14/15/16）は SPI デバイス（SD カード等）と同時使用不可
- アナログ兼用ピン（18〜21）は ADC として使う場合は `analogRead()` と競合するため注意
- Pro Micro はリセットボタンがないため、書き込み時は RST ピンを一時的に GND に落とす
- 5V モデルと 3.3V モデルで PID が異なる
  - 16MHz/5V: VID=0x1B4F, PID=0x9206
  - 8MHz/3.3V: VID=0x1B4F, PID=0x9204
