# ピンアサイン

対応ボード:
- [Pro Micro](#ボード-sparkfun-pro-micro-atmega32u4-16mhz-5v)
- [UIAPduino](#ボード-uiapduino-pro-micro-ch32v003)

ボード固有の設定は `src/boards/<ボード名>/board_config.h` に定義されています。
`platformio.ini` の `-I` フラグでビルド対象のボードが選択されます。

## マトリクス配線の仕組み

スイッチは Row/Column マトリクスで接続する。
各スイッチは 1 本の Row ラインと 1 本の Col ラインの交点に配置する。

- **Row ピン**: OUTPUT（スキャン時に 1 本ずつ LOW に駆動）
- **Col ピン**: INPUT_PULLUP（押下時に LOW を検出）
- **ダイオード**: 各スイッチにダイオードを直列に入れ、ゴースト押下を防止する

スキャン手順:
1. 全 Row を HIGH に保つ（非選択状態）
2. 対象 Row を LOW に駆動
3. 各 Col ピンを読み取り — LOW なら押下、HIGH なら非押下
4. Row を HIGH に戻す
5. 次の Row へ

ボタン番号 = row × COL_COUNT + col（0 始まり）

## 注意事項

- マトリクス配線では各スイッチにダイオードを入れること（NKRO 対応・ゴースト防止）
- Row/Col ピンの割り当ては `board_config.h` で変更可能
- **Pro Micro**: リセットボタンがないため、書き込み時は RST ピンを一時的に GND に落とす
- **UIAPduino**: リセットボタンあり。書き込み待機モードへの入り方やシームレススイッチ機能については [flash_procedure.md](flash_procedure.md) を参照
- Pro Micro の 5V モデルと 3.3V モデルで PID が異なる
  - 16MHz/5V: VID=0x1B4F, PID=0x9206
  - 8MHz/3.3V: VID=0x1B4F, PID=0x9204
